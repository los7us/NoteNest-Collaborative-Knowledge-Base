# NoteNest Documentation

Welcome to the **NoteNest documentation** 
This folder contains all non-code documentation related to the project.

Documentation is a **first-class contribution** in NoteNest.  
You do NOT need to write code to contribute here.

---

## What Is This Folder For?

The `docs/` folder is used to:
- Explain how the project works
- Help contributors get started
- Document APIs and architecture
- Describe roles, permissions, and workflows

Well-written documentation helps:
- New contributors onboard faster
- Mentors review contributions easily
- The project scale sustainably
