##  What Is Tested?

Backend tests focus on:
- API behavior
- Authentication flows
- Role-based access control (RBAC)
- Error handling
- Edge cases

---

##  Folder Purpose

backend/tests/


This folder is used for:
- Writing automated tests
- Documenting test cases
- Reproducing bugs in a testable way

---

##  Beginner-Friendly Contributions

You can contribute by:
- Writing simple test cases
- Adding test descriptions (even without code)
- Documenting expected API behavior
- Reporting missing test coverage

Example beginner tasks:
- Test that unauthenticated users cannot access notes
- Test that viewers cannot edit notes
- Test invalid input handling
